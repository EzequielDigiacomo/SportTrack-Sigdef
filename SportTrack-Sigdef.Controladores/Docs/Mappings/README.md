# Mapeos de Clases y DTOs - SportTrack-Sigdef.Controladores

Esta carpeta contiene los archivos de documentación con la equivalencia de campos entre las **Entidades de Dominio** y los **DTOs** (Data Transfer Objects) utilizados para las llamadas a la API.

El objetivo de estos mapeos es servir de referencia rápida para evitar errores de coincidencia de nombres o tipos al modificar campos en el backend.
